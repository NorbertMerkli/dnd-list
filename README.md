# React Starter Template

A modern, production-ready React 19 starter template featuring the React Compiler, Vite, TypeScript, and accessibility-first UI libraries.

## Tech Stack

| Category | Technology |
|----------|------------|
| Build | Vite 8 |
| Framework | React 19 + React Router 7 |
| Language | TypeScript |
| Styling | TailwindCSS 4 with `@tailwindcss/vite` |
| UI Components | React Aria + React Aria Components + TailwindCSS React Components |
| State Management | React Stately |
| Data Fetching | TanStack Query 5 |
| Animations | Motion |
| Internationalization | @internationalized/date, @internationalized/number |
| Compilation | React Compiler (babel-plugin-react-compiler) |

> **Note**: This template leverages the React Compiler for automatic optimization of component re-renders and fine-grained reactivity.

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |
